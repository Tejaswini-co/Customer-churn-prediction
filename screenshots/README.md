2. For each visualization cell, right-click on the output and select "Save as..."
3. Save as PNG files in this folder
4. Update the Medium article with actual image links

## Windows Screenshot Tips:

- Use Windows + Shift + S for snipping tool
- Or use Snip & Sketch app
- Save as PNG format for best quality
- Ensure plots are clearly visible with legends
